## 整体风格
The visual style follows a retro-futuristic pixel art aesthetic with modern UI polish. Dark navy backgrounds (#1A1A2E, #16213E) provide high contrast for bright accent colors (primary blue #4A90E2, error red #E74C3C). Typography uses monospace/pixel fonts to reinforce the retro gaming feel. All interactive elements feature subtle glow effects and crisp 2-4px borders. Error states are prominently highlighted with red borders and warning icons to ensure immediate user attention when critical failures occur (like corrupted game files).

## 组件视觉规范

# Component Visual Specifications

## Error Modal / Alert Dialog

**Purpose**: Display critical error messages when game fails to load (e.g., corrupted index.html)

**Layout**:
- Width: 480px (max-width: 90vw)
- Height: auto, min-height: 240px
- Position: center viewport with overlay
- Padding: 24px (lg)

**Visual Style**:
- Background: `bg_secondary` (#16213E)
- Border: 2px solid `error` (#E74C3C)
- Border radius: `md` (4px)
- Box shadow: `lg` + `glow` (error red glow)
- Overlay: `overlay` (rgba(0, 0, 0, 0.8))

**Typography**:
- Title: `xl` (24px), `text_primary`, bold, margin-bottom: `md`
- Message: `base` (14px), `text_secondary`, line-height: 1.6
- Error code: `sm` (12px), `text_disabled`, monospace

**Icon**:
- Error icon: 48px, `error` color, positioned top-left or centered above title
- Style: outline or filled warning triangle

**Button (Reload/Retry)**:
- Background: `primary` (#4A90E2)
- Text: `text_primary`, `md` (16px)
- Padding: 12px 24px
- Border radius: `sm` (2px)
- Hover state: brightness +10%, shadow `md`
- Active state: brightness -10%

**States**:
- Default: as described
- Hover (button): background lightens, cursor pointer
- Disabled (button): opacity 0.5, cursor not-allowed

## Loading Screen

**Purpose**: Display while game assets are loading

**Layout**:
- Full viewport (100vw x 100vh)
- Centered content

**Visual Style**:
- Background: `bg_primary` (#1A1A2E)
- Loading spinner: 64px, `primary` color with `glow` shadow
- Progress bar (optional): width 320px, height 8px, `bg_secondary` background, `primary` fill

**Typography**:
- Loading text: `lg` (20px), `text_primary`, centered below spinner
- Tip text (optional): `sm` (12px), `text_secondary`, margin-top: `lg`

**Score Display**:
